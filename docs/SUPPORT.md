# Support

NativeFrame UI is an open-source project. Public questions, reproducible bugs, and documentation gaps should use GitHub Discussions or Issues as appropriate.

When asking for help, include:

- Windows version and architecture.
- Visual Studio, Windows SDK, and CMake versions.
- Debug or Release preset.
- A minimal reproduction or sample.
- Relevant logs and exact error messages.

Commercial teams may use the MIT-licensed core without a support contract. Optional commercial services can include integration assistance, custom controls, migration work, compatibility testing, and prioritized support; these services are separate from the library license.
